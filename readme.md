# SeAl Pro – Quantum ESPRESSO GUI

## Features
- CIF / XYZ upload
- 3D visualization
- SCF / Relax / VC-Relax / Bands
- Spin-polarized calculations
- DFT+U
- Hybrid functional (HSE)
- Automatic k-mesh
- Slab builder
- Band structure plotting
- DOS plotting
- Phonon input generator

## Run

pip install -r requirements.txt
streamlit run app.py
